# Chat Notes 💬

A real-time anonymous chat application built using React and Firebase, enabling users to send and receive messages instantly across multiple devices.

## 🚀 Live Demo

https://my-chat-notes.web.app/

## 🛠️ Tech Stack

* React (Frontend)
* Firebase Realtime Database (Backend)
* Firebase Authentication (Anonymous Auth)
* Firebase Hosting (Deployment)

## ✨ Features

* Real-time messaging across multiple users
* Anonymous authentication (no signup required)
* Unique user session identification
* Username support
* Timestamp for each message
* Responsive full-screen chat UI
* Auto-scroll to latest messages

## ⚙️ How It Works

* Users are authenticated anonymously using Firebase
* Messages are stored in Firebase Realtime Database
* Real-time listeners sync messages instantly across all clients
* No backend server required (serverless architecture)

## 📦 Installation

```bash
git clone https://github.com/SelvaAjeethS/chat-notes.git
cd chat-notes
npm install
npm start
```

## 🚀 Deployment

```bash
npm run build
firebase deploy
```
